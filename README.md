<!-- ===================== SOBRE ===================== -->
<h2>📌 Sobre o Projeto</h2>

<p>
O <strong>English Quest</strong> é uma plataforma educacional gamificada voltada ao ensino de inglês em escolas públicas.
O sistema utiliza <strong>minigames interativos</strong>, <strong>progressão</strong> e <strong>exploração</strong> para tornar o aprendizado mais envolvente e eficaz.
</p>

<ul>
<li>🎮 Jogo acessível via navegador</li>
<li>🧑‍🏫 Painel para professores</li>
<li>📊 Coleta de dados anonimizados</li>
<li>🔐 Foco em privacidade (sem dados pessoais)</li>
</ul>
